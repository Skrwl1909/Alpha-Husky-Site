Planned Progression System

Instinct Tree
Long-Term Build Identity

Instinct Tree is a planned post-launch progression layer designed to deepen player identity across combat, crafting, pets, and faction play — without replacing the importance of gear, forging, or the world itself.

The goal is specialization, not power creep. It expands the game’s existing loops instead of competing with them, giving players a stronger build path inside the world they are already shaping.

NFT Direction

World-Linked Collectibles
Premium First. Utility With Restraint.

If Alpha Husky moves into NFTs, the strongest path is not chart support or combat advantage. It is premium identity: cosmetics, prestige, collectibility, and world-linked supporter presence that feels native to the game.

Recommended Start

A small, highly polished Founder / Supporter Relics collection with effective ownership unlocks — skins, frames, badges, and subtle prestige marks that stay active while the wallet still holds the NFT.

Model A — Safe Premium

Cleanest first step: premium cosmetics, prestige, and a mature collector identity before token complexity grows.

Model B — Growth Collector

Later expansion into faction, event, or seasonal sets that reward long-term collecting without turning Alpha into a drop machine.

Model C — Dynamic Premium

Long-term vision: NFTs that evolve with milestones, factions, and world events once the base product is fully stable.

No pay-to-win
No price promises
Cosmetics & prestige first
Effective ownership unlocks
Direction Card: The most reasonable start is a single premium collection grounded in the Alpha world. Collector sets can come later. Truly dynamic NFTs should only arrive once the world has earned them.
